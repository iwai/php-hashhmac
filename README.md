
## Usage

```
Usage: php php-hashhmac.phar [--help|-h] [--verbose|-v] [--algorithm|-a <algorithm>] [--key|-k <key>] [<file>]

Examples

./bin/php-hashhmac.phar -k YOUR_KEY -f message.json
./bin/php-hashhmac.phar -a sha256 -k YOUR_KEY -f message.json
```
